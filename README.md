# Diabetes Prediction using Machine Learning

## Project Overview
This project predicts whether a person is diabetic or not using a Machine Learning model (SVM).

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Features
- Data preprocessing
- Feature scaling using StandardScaler
- Model training using SVM
- Accuracy evaluation
- Prediction system for new input

## Dataset
PIMA Indians Diabetes Dataset

## How to Run
1. Install required libraries
2. Run the Python file
3. Enter input values to get prediction

## Output
- 0 → Not Diabetic  
- 1 → Diabetic
